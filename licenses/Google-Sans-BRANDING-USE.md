# Google Sans — Sign in with Google branding use

This font is bundled only for the localized **Sign in with Google** button. It is not part of the Gigagochi application typography and must not replace OpenRunde elsewhere.

Official provenance, verified 2026-07-17:

- Branding requirement: <https://developers.google.com/identity/branding-guidelines?hl=en>
- Official specimen linked by the branding page: <https://fonts.google.com/specimen/Google%2BSans>
- Official CSS request: <https://fonts.googleapis.com/css2?family=Google+Sans:wght@500&display=swap>
- CSS-resolved TrueType source: <https://fonts.gstatic.com/s/googlesans/v69/4Ua_rENHsxJlGDuGo1OIlJfC6l_24rlCK1Yo_Iqcsih3SAyH6cAwhX9RFD48TE63OOYKtrw2IKli.ttf>
- SHA-256: `c6cfaf61002644fa30db574d45068fa8e4427f847c4d1659c7a023df9285cb20`
- Embedded family/style: `Google Sans, Google Sans Medium` / `Medium, Regular`

The official Google Identity page requires Google Sans Medium 14/20 for a custom light Sign in with Google button and permits localization of the call-to-action. This repository uses the font only within that branding context. Google brand assets are not covered by the OpenRunde SIL OFL license.
